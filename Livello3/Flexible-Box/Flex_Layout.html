Flex Layout Box Model

Un flex container è un box generato da un elemento che ha proprietà display uguale a flex o inline-flex

flex: produce in un elemento la creazione di un box-flex di blocco
inline-flex: produce in un elemento la creazione di un box-flex inline

I valori accettati da flex (o inline-flex) sono:

none: equivalente a 0 0 main-size (width).
auto: equivalente a 1 1 main-size
flex-grow:
flex-shrink:
flex-basis:

  • float e clear

flex-flow

La proprietà flex-flow permette di definire contemporaneamente le proprietà flex-direction e flex-wrap 

element { 
	flex-flow: <'flex-direction'> || <'flex-wrap'>          
}

Flex-direction

La proprietà flex-direction permette di definire come gli elementi del flex container devono essere piazzati. I valorei accettati sono quattro:

row: sistema gli elementi nel modello inline cioè uno di fianco all'altro
row-reverse: come row con la differenza che li inverte
column: sistema gli elementi nel modello blocco cioè uno sotto l'altro
column-reverse: come column con la differenza che li inverte

Container { flex-direcion: row; }
flex: auto

Flex-wrap

La proprietà flex-wrap permette di definire se gli elementi del flex container debbano rimanere su una unica linea (blocco) o più linee. I valori possibili sono tre:

nowrap: il flex container è una unica linea
wrap: il flex container è multilinee
wrap-reverse: come wrap con la differenza che l'ultimo elemento si scambia di posto col primo

Flex Lines

Il flex container, come visto sopra, può essere creato in una sola linea o più linee a seconda dell'impostazione nowrap o wrap. Con impostato wrap si ha la multilinea.
Nell'esempio di sopra con ( flex-flow: row wrap ) si ha che il quarto elemento del flex container si vada a posizionare in una seconda linea perchè la larghezza totale degli elementi è superiore a quella del loro genitore. La larghezza (width) degli elementi non viene influenzata.
