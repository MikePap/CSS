flex-grow, flex-shrink e flex-basis

I valori flex-grow, flex-shrink e flex-basis sono accettati dalla proprietà flex ( o inline-flex ) per ogni singolo elemento per determinare o modificare la sua dimensione di 'width'.
In altre parole se ( oltre al genitore ) definiamo la proprietà flex per un singolo elemento del flex container lo si può condizionare ad assumere valori differenti rispetto a quelli definiti dalla proprietà width.
E' consigliato utilizzare la shorthand flex per impostarli tutti i tre piuttosto che singolarmente: 

flex: 1 1 0%;			sono i valori di default di tutte e tre 
è come se si scrivesse --> flex-grow: 1, flex-shrink:1, flex-basis:0%

flex-grow

Permete di determinare di quanto un elemento del flex container deve crescere relativamente agli altri elementi del flex container quando c'è dello spazio eccedente

Nell'esempio seguente sono impostati i valori default di tutti e tre : flex: 1 1 0%;. Succede che la larghezza si espande fino a raggiungere quella del genitore. Anche se s'imposta una 'width', non sarà tenuta in considerazione. Non cambia niente anche con flex-basis ad 'auto' oppure flex: 1 0 0.

div genitore width: 400px
elemento

Lo stesso effetto lo abbiamo se impostiamo: flex: 1 1 auto oppure flex:1 0 auto o flex: auto o flex: 1

Ora impostiamo la flex-grow a valore 0 e il flex-basis a 0 ( flex: 0 0 0 ). Succede che l'elemento si allarga per contenere il testo all'interno di esso (questo funziona solo in Firefox; negli altri browser l'elemento non viene visualizzato).

Si desume quindi che il valore della flex-grow è relativo alla flex-basis che se impostata ad auto influenza la crescita della width

Nei seguenti esempi si impostano dei valori differenti di flex-grow e flex-basis per ogni elemento.

2 0 100px
0 0 80px

2 0 100px

Si notano delle differenze dal primo al secondo esempio. Per capire come viene distribuito lo spazio ci vengono in aiuto le seguenti formule

Spazio Disponibile = [larghezza container] - [somma dei flex-basis di ogni elemento]
Unità di ingrandimento = [Spazio Disponibile] / [somma dei flex-grow di ogni elemento]
Grandezza elemento = [flex basis] + [(Unità di ingrandimento) x (flex-grow)]

Applicandole al secondo esempio avremo:

Spazio Disponibile = 400 - 280 = 120 (dove 280 => 100 + 80 + 100)
Unità di ingrandimento = 120 / 3 = 40 (dove 3 => 2 + 0 + 1)
grandezza elemento 1 = 100 + (40*2) = 180 px
grandezza elemento 2 = 80 + (40*0) = 80 px (non c'è aumento)
grandezza elemento 3 = 100 + (40*1) = 140 px

flex-shrink

Permette di determinare quanto un elemento del flex container può restringersi relativamente agli altri elementi del flex container quando c'è dello spazio eccedente.

Nel primo esempio poniamo flex: 0 1 auto; Si nota che tutti e tre gli elementi, che hanno una larghezza di 200px, si restringono ugualmente per rientrare nello spazio del genitore che è di 400px

Rendere gli elementi multipli uno dell'altro

Se si vuole rendere gli elementi di un contenitore flex multipli uno dell'altro si può agire solo sul flex-grow di ognuno lasciando a zero gli altri

3 0 0
1 0 0
2 0 0

Si ha che il primo elemento che ha flex-grow 3 (giallo) sarà il triplo dell'elemento 2 (rosso ) ed una volta e mezzo del terzo elemento (verde)
