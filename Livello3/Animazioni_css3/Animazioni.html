Animazioni con CSS3

Al momento funziona in tutti i browser. In IE dalla versions 10; soltanto per Chrome c'è bisogno della parola chiave webkit.

Le animazioni con i CSS3 vanno create all'interno della regola @keyframes. Dopodichè è possibile richiamare tali impostazioni in qualsiasi elemento tramite la parola chiave animation.

Nella parola chiave animation si possono racchiudere tutte le proprietà delle animazioni ad eccezione della proprietà animation-play-state. 

animation: name duration timing-function delay iteration-count direction fill-mode;

Le proprietà delle animazioni sono le seguenti e vanno specificate nel seguente ordine:

animation-name: specifica il nome dell'animazione creata nella regola @keyframes
animation-duration: specifica il tempo di durata dell'animazione che può essere spresso in secondi(es:3s). Il valore di default è 0.
animation-timing-function: specifica l'andamento dell'animazione. Si possono utilizzare dei valori predefiniti che sono:
linear: l'animazione ha la stessa velocità dall'inizio alla fine
ease: che è il valore di default ha una velocità più lenta all'inizio e alla fine e più forte nel mezzo
ease-in: l'animazione ha una partenza lenta per poi aumentare velocità
ease-out: al conrario di ease-in ha una finale più lento
ease-in-out: ha una partenza ed un finale più lento; sembra uguale a ease
cubic-bezier(n,n,n,n): è possibile specificare i desiderati valori impostando dei valori da 0 ad 1
animation-delay: è opzionale e specifica il tempo di ritardo dell'inizio animazione in secondi come per animation-duration
animation-iteration-count: specifica quante volte un animazione deve essere ripetuta. Lo si può indicare impostando un valore numerico oppure la si può impostare con infinite che sottointende che non deve avere fine
animation-direction: specifica se l'animazione può avere un senso alternato; i due valori impostabili sono:
normal: che è il valore di default sottointende che l'animazione ha un percorso normale
alternate: definisce un che l'animazione debba rifare il percorso anche in senso opposto cioè al contrario del suo senso iniziale
animation-fill-mode: specifica la posizione che l'elemento assumerà a fine animazione o se l'animazione viene interrotta. (vedi qui)
none: l'animazione non applicherà nessuno stile
forwards: a fine animazione l'elemento animato rimarrà posizionato nel punto finale dell'animazione(simile a 'fill=freeze' nei SVG)
backwards: a fine animazione l'elemento animato rimarrà posizionato nel punto iniziale dell'animazione
both: ....
inherit: eredita la proprietà dal genitore
animation-play-state: può fermare e far riprendere l'animazione impostando rispettivamente i seguenti valori: paused e running

Ognuna delle suddette proprietà descritte ha una sua sintassi per Javascript:
object.style.animation="mia_animazione 5s infinite"
object.style.animationName="mia_animazione"
object.style.animationDuration="3s"
object.style.animationTimingFunction="linear"
object.style.animationDelay="2s"
object.style.animationIterationCount=3
object.style.animationDirection="alternate"
object.style.animationFillMode = "forwards"
object.style.animationPlayState="paused"
Tutte le proprietà in unica dichiarazione: animation: test 1s 2s 3 alternate backwards;

Prefissi browser

#box {
	-webkit-animation: NAME-YOUR-ANIMATION 5s infinite; /* Safari 4+ */
	-moz-animation:    NAME-YOUR-ANIMATION 5s infinite; /* Fx 5+ */
	-o-animation:      NAME-YOUR-ANIMATION 5s infinite; /* Opera 12+ */
	animation:         NAME-YOUR-ANIMATION 5s infinite; /* IE 10+ */
}

Molteplici step

@keyframes fontbulger {
	0% {
		font-size: 10px;
	}
	30% {
		font-size: 15px;
	}
	100% {
		font-size: 12px;
	}
}

#box {
	animation: fontbulger 2s infinite;
}
quando invece ci sono dei valori uguali in uno o più step su usa la virgola 
@keyframes fontbulger {
	0%, 100% {
		font-size: 10px;
	}
	50% {
		font-size: 12px;
	}
}

Combinazione animazione con "transform" 

@keyframes infinite-spinning {
	from {
		transform: rotate(0deg);
	}
	to {
		transform: rotate(360deg);
	}
}

Animazioni multiple

Vedere esempio nel file Animazione_04.html 

.animate-this {
	animation: first-animation 2s infinite, 
	another-animation 1s;
}

steps()

La funzione steps() stabilisce quanti keyframe inserire nello spazio di tempo dell'animazione (vedere il file 'Animazione_05.html'). 

@keyframes move {
	from{ top: 0; left: 0; }
	to{ top: 100px; left: 100px; }
}

.move {
	animation: move 10s steps(10) infinite alternate;
}
