box-shadow

Creazione di ombre sugli elementi. Internet Explorer è l'unico browser che non supporta box-shadow. Per Firefox c'è bisogno del prefisso "-moz-" e per Safari e Chrome di "-webkit-"

box-shadow:[inset(optional)] -- [value(x-coordinate offset)] -- [value(y-coordinate offset)] -- [value(shadow blurriness)] -- [value(color)]

Il valore inset è opzionale e se specificato indica che l'ombra sarà interna all'elemento
Il primo valore fa riferimento alle coordinate offset left or right(per il right il valore deve essere negativo)
Il secondo valore fa riferimento alle coordinate offset top or bottom(per il top il valore deve essere negativo)
Il terzo valore è opzionale e specifica il valore della opacità dell'ombra
Il quarto valore specifica il colore dell'ombra
box1=> -8px 10px 0 grey
box2 => 8px 8px 12px black
box3 => inset 8px 8px 8px grey
box4 => -10px -10px 30px red
box5 => 15px -10px 15px black
border => 2px solid bianco
box6 => inset -8px -18px 12px grey
border => 1px solid giallo
